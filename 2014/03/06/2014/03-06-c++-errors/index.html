C++ Primer 中文版第五版错误（持续更新中）

#C++
Mar 6, 2014

这篇文章是对 《C++ Primer 中文版第5版》 中错误的列举，会随着时间的推移持续更新

  1. P16：练习 1.19，“修改你为 1.4.1 节练习 1.10” 改为 “修改你为 1.4.1 节练习 1.11”
  2. P21：1.6 书店程序中，第5行的注释 // 保存下一条交易记录的变量 和第8行的注释 // 保存和的变量 互相交换位置
  3. P33：中间位置第 5 个点号 “当我们赋给无符号类型一个超出它表示范围的值时，结果是…” ，见C++ 中有符号类型到无符号类型的转换
  4. P38: 练习 2.6， int month = 09，错误，没有八进制的 9
  5. P50: “引用（reference）为对象起了另外一个名字”应该改为“引用（reference）为对象取了另外一个名字”
